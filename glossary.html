Find Kit Skus

  1. Enter View Product Activity
  2. Enter sku for single item if known, else click magnifying glass
  3. Click action button (3 dots)
  4. Select search related collection
  5. Scroll down to included products and only select include kit masters
  6. Fill in any other relavent information
  7. Click search
  8. Check boxes next to the options desired
  9. Push Save
  10. Bottom left box shows availability
  11. Select arrow next to net available and select a number greater than One
  12. Click off of the box, if kits are available, view their location

Multiple Deliveries

  1. Navigate to order fulfillment screen inside Order entry
  2. Click New Fulfilment and accept that another delivery already exists.
  3. Double click route code for new fullfilment.
  4. Add Fulfilment Method for each one(click the arrow button between selections) and select date
  5. Click the lines button at the bottom
  6. Click the boxes to the left of the items for each fulfilment, push the arrow after each one. **You can verify the fulfilment info for each fulfilment in the middle/upper fullfilment box
  7. If you get an alert that you can't change the items on a fulfillment on a particular selection. Push the left arrow next to fullfilments to go to the first one. It will now let you.
  8. Once you've selected items for all the available fullfilments, Click the exit button. it will prompt you t o save or discard the changes. PUSH NO TO SAVE THE SELECTIONS
  9. Click the exit button

Entering Multiple salespeople

  1. When inside a ticket and prompted to add a salesnumber do the following:
  2. Type an associate number in the salesperson field or push the action button (3 dots) and click "Multiple Commision Entry" to enter additional salespeople.
  3. Enter additional associate numbers and then press enter or the search icon.
  4. Push the green plus to add or the blue circle with the arrow to change the salesperson that you want to add. * you can also search for salespeople with the magnifying glass.
  5. If you want to remove a salesperson then douple click a salespersons name and click the red minus button.
  6. click save when complete.
  7. Click ok when it asks to split the commision evenly
