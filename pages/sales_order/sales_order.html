Write a Sales Order

  1. Open "Enter a Sales Order"
  2. Click the Green Button
  3. Select order type: sales order or Sales Quote
  4. Input Sales Person Number
    1. Type an associate number in the salesperson field or push the action button (3 dots) and click "Multiple Commision Entry" to enter additional salespeople.
    2. Enter additional associate numbers and then press enter or the search icon.
    3. Push the green plus to add or the blue circle with the arrow to change the salesperson that you want to add. * you can also search for salespeople with the magnifying glass.
    4. If you want to remove a salesperson then douple click a salespersons name and click the red minus button.
    5. click save when complete.
    6. Click ok when it asks to split the commision evenly
  5. Search for a customer by phone number, if unknown, click the magnifying glass and fill out the known info on the left. Push search and double click the customer
  6. Select Merchandise Tab on Left Side
  7. Enter SKU under product and push tab or click the magnifying glass . if unknown then select
    1. Product field is only used for SKU's
    2. Description has to match the buyers description and word order
    3. Vender is the short vender code ex: MND, PAHO, SHTF and etc. This must be capitalized
    4. Click Search
    5. Check boxes to the left of product on all items that the customer wants
    6. Click Save
  8. Select the quanity of the item that the customer wants
  9. Input all individual coupon codes
  10. Select AS is button if it is as is. (You can search for it if needed)
  11. Select save and add another to add another item to ticket and repeat previous steps or select Save to move on.
  12. Modify Order ** if you only see SKU numbers then do as follows:
    1. Left click above item field
    2. A drop down should appear
    3. Select the box next to description
    4. Deselect extended price
    5. Click off of the drop down bar
    1. Remove Item - click the red remove button next to item
    2. Adjust Quantity - double click on the item and change quanitity field and then click save.
  13. Select Fulfilment on left side of screen (exit the fab screen)
  14. Select Fulfilment date: if unsure, select "customer will call" in the status field
  15. Select Fulfilment type from Method: Delivery, Customer take with, Customer Pick Up
  16. under handling method: select if there are special notes for delivery. (leave in box or etc ) This is rare.
  17. Verify Address: if wrong, click 3 dots and fill out info and click save
  18. If multiple deliveries are required follow
    1. Click New Fulfilment and accept that another delivery already exists.
    2. Double click route code for new fullfilment.
    3. Add Fulfilment Method for each one(click the arrow button between selections) and select date
    4. Click the lines button at the bottom
    5. Click the boxes to the left of the items for each fulfilment, push the arrow after each one. **You can verify the fulfilment info for each fulfilment in the middle/upper fullfilment box
    6. If you get an alert that you can't change the items on a fulfillment on a particular selection. Push the left arrow next to fullfilments to go to the first one. It will now let you.
    7. Once you've selected items for all the available fullfilments, Click the exit button. it will prompt you t o save or discard the changes. PUSH NO TO SAVE THE SELECTIONS
    8. Click the exit button
  19. Click the payment button on the left side of the screen
  20. Discount code field: Use the scroll down box to select any discounts.
  21. Push the 3 dots in the deposit section to Pay with card.
    1. Click the magnifying glass and double click yhe vender type
    2. Type in card number
    3. Select month and year from drop down box and put in cvv number and amount
  22. Push the 3 dots in the finance section to Pay with credit
  23. Push Save
  24. Select the printer
  25. Choose your print options (by fulfilment if there are multipe fullfilments or by room if you have multiple rooms assigned
